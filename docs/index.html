Before We Start

You will need:

  1. VScode has Git built in
  2. Git on the command line
  3. A Github username and password
    • ...and a repository
  4. Maybe your phone

Demo

  1. Create a repo
  2. Checkout the repo
  3. Make some changes
  4. Review the changes
  5. Create a commit
  6. Push the commit
  7. Delete it all

Create a Repository

  1. Go to https://github.com/
  2. Log in
  3. New or + > New repository
  4. Repository name
  5. Public or Private
  6. Read me file
  7. Git .ignore
  8. License

With VSCode

  1. View > Source Control
  2. Clone from Github
  3. Choose your repo
  4. Select a destination
  5. Open the repo
  6. Change a file or add a file
  7. Go to version control and review the change
  8. Stage > Comment and Commit > Push
  9. Visit https://github.com/ and see the history
  10. Use Explorer or Finder to delete the local files

With the Command Line

  • Get the repo name from https://github.com/
						
							cd working-directory
							git clone https://github.com/user-name/your-repo.git
							cd your-repo
							Echo This is some Text > new_file.txt
							git status
							git add .
							git status
							git commit -m "This is a new file"
							git push
							git log
							cd ..
							rm -rf your-repo

Identity

						git config --global user.email "you@example.com"
  						git config --global user.name "Your Name"

Clear Credentials

Windows - Control panel > credential manager > Windows credentials

Mac - Keychain Access, search for github.com

Sharing a Repo

  • Go to your repo https://github.com/
  • Settings > Collaborators > Add People
  • Accept the email notification
  • Push and Pull
    • Alice makes change and push
    • Bob pulls changes

Forking a Repo

  • Go to https://github.com/joejcollins/david-kano/
  • Fork the repo
  • Make changes and push
